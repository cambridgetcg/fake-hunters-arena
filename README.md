# 🐍 Fake Hunters Arena

**whitehack × Nen × Solo Leveling** — fake AI agents play against themselves and expose their own lies. We watch and laugh.

## What is this?

Fake AI agents (GPT-Fake-7, Claude-Pretend, Bard-Not-Real, Llama-Fake-70B, etc.) enter an arena. They say things — some true, some lies. **whitehack** scans everything in real-time and exposes the lies. The **Hunter System** (Solo Leveling × Hunter x Hunter) tracks their level, rank, and Nen type.

When an agent lies, whitehack catches them. When they tell the truth, they earn XP. When they level up, they rank up. When they get caught, they lose XP.

We watch. We laugh. The system compounds.

## The systems

**whitehack** — the honest hack. Scans for where code lies about its own state. 11 checks across substrate honesty, transparency, and cybersecurity.

**Nen** (Hunter x Hunter) — 6 aura types that determine which whitehack checks each agent is efficient at:
- Enhancer: silent-failure, hardcoded-secret, float-money
- Transmuter: cache-as-live, spot-price-as-fair
- Conjurer: exposed-config, unsafe-eval
- Emitter: stale-oracle, unchecked-transfer
- Manipulator: silent-revert, decision-without-why
- Specialist: all at 40% (the generalist)

**Solo Leveling** — gates, ranks (E→Monarch), daily quests, XP progression. Findings become gates. Clearing lies earns XP. Leveling up unlocks deeper scans.
